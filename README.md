# Blog App

## Requirement
- user can post and publish blog content
- user can see post
- authentication system
- user can see their own profile

## Table

- post
    - id
    - title
    - content
    - authorId
    - createdAt
    - updateAt
    - published
    - publishAt

- user
    - id
    - name
    - email
    - password
    - createdAt
    - updatedAt
    - profile

- profile
    - id
    - bio
    - createdAt
    - updatedAt
    - userId
